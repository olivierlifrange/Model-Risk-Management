Project Objective: This project aims replicates a real-world Model Risk Management (MRM) workflow by comparing regulatory-standard "Champion" models against an ML-driven "Challenger" model.

The Champion Models (Baseline): uses historical Simulation and GARCH models aiming to replicate industry-standard baseline for Value at Risk (VaR).

The Challenger Model (XGBoost): uses time-series features to train an XGBoost model, aiming to capture non-linear market dynamics that traditional models might miss.
